### 经典输入方式
#### 第一行给 T 组数据（竞赛最常见）
第一行输入一个整数 T，表示有 T 组测试数据
```
3
1 2
3 4
5 6
```

```
#include <iostream>
using namespace std;

int main() {
    int T;
    cin >> T;

    while (T--) {
        int a, b;
        cin >> a >> b;
        cout << a + b << endl;
    }
}
```

#### 读到 EOF（ACM 经典模式）
题目会说多组输入，直到文件结束
```
1 2
3 4
5 6
```

```
#include <iostream>
using namespace std;

int main() {
    int a, b;

    while (cin >> a >> b) {
        cout << a + b << endl;
    }
}
```


### 如果想读取字符串

```
hello world
```


```
string s;

//下面这两个一定要绑定在一起，因为一定要去除之前一行的换行，一定要死记硬背
cin.ignore();       //忽略一个字符，可以是\n
getline(cin, s);    //getline的逻辑是一直读直到遇到 \n，换句话说就是读取一行
```


```
输入：
3
9
one space
11
two  spaces
14
three   spaces


输出：
ecapseno
secapsowt
secapseerht
```

```
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t;
    cin >> t;

    while(t--){
        int n;
        cin >> n;
        cin.ignore();  // 清掉上一行换行
        string s;
        getline(cin, s);    //这两个是绑定在一起的
        string ans;
        for(char c : s){
            if(c != ' ')
                ans += c;
        }
        reverse(ans.begin(), ans.end());
        cout << ans << endl;
    }

}
```


### printf输出控制格式（针对小数等情况）

```
printf("格式控制字符串", 变量1, 变量2, ...);

%d      → int
%lld    → long long
%f      → double
%.kf    → 保留 k 位小数
%nd     → 输出宽度至少为 n,默认右对齐,左侧补空格
%0nd    → 补零输出，
%s      → 字符串

```
