```
二分查找的模版
1、循环条件是while(left <= right)，直到right < left
2、right   指向最后一个 < target 的位置      left 指向第一个 ≥ target 的位置(这个要死记硬背的！！！)

相当于这是，这是 C++ 的 lower_bound 手写版
```

```
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0;
        int right = nums.size() - 1;
        int mid;

        while(left <= right){
            mid = left + (right - left) / 2;
            if(nums[mid] == target)     return mid;
            else if(nums[mid] > target) right = mid - 1;
            else left = mid + 1;
        }

        return left;

    }
};
```
