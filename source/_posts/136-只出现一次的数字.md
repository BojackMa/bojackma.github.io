关于引用和指针
```
vector<int> nums = {1,2,3};  
vector<int> v(5);  // 大小为5，所有元素默认初始化为0
vector<int> v(5, 10);  // 大小为5，每个元素都是10
int singleNumber(vector<int>& nums) //引用传递
```


```
class Solution {
public:
    {
        int ret = 0;
        for(auto it = nums.begin(); it != nums.end(); it++){
            ret = ret ^ *it;
        }
        return ret;
    }
};

//要记住位运算符号是按位异或^、按位与&、按位或|、按位取反~、左移 <<、右移 >>
//hash表也能做，但是不符合空间要求
//异或，异或可以做到相同的两者相消，并且异或满足交换律，即a异或b异或a等于b，相当于筛选出b来了，本质上还是位运算。
```
